# es6-proxies
Source for Nodejs meetup talk 

## Getting started

### clone the project

```bash
git clone https://github.com/Khaledgarbaya/es6-proxies.git
```

### Install dependencies

Using yarn 

```bash
yarn
```
using npm 

```bash
npm install
```

### run the project

```bash
npm run test-final:watch
```
